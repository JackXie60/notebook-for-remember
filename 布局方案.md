# 布局
## 多栏布局
**两栏布局**
```
<style>
        .clearfix::after{
            content: "";
            clear: both;
            display: block;
        }
        .container{
            background-color: orange;
            width: 90%;
            margin: 0 auto;
        }
        .aside{
            width: 300px;
            height: 300px;
            float: left;
            background-color: orchid;
        }
        .main{
            height: 300px;
            overflow: hidden;
            background-color: palegreen;
        }
    </style>
</head>

<body>
    <div class="container clearfix">
        <div class="aside">

        </div>
        <div class="main"></div>
    </div>
</body>
```
**三栏布局**
```
<style>
        .clearfix::after{
            content: "";
            clear: both;
            display: block;
        }
        .container{
            background-color: orange;
            width: 90%;
            margin: 0 auto;
        }
        .left{
            width: 300px;
            height: 300px;
            float: left;
            background-color: orchid;
        }
        .right{
            width: 300px;
            height: 300px;
            float: right;
            background-color: green;
        }
        .main{
            height: 300px;
            overflow: hidden;
            background-color: palegreen;
        }
    </style>
</head>

<body>
    <div class="container clearfix">
        <div class="left"></div>
        <div class="right"></div>
        <div class="main"></div>
    </div>
```
