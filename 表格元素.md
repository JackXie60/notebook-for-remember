# 表格元素
在css技术出现之前，网页通常使用表格来进行布局

表格不再使用网页布局?
>表格的渲染速度过慢，只有后台会用

标题单元格：th
普通单元格：td
```
<table>//display:table
    <caption>标题</caption>
    <thead>
        <tr>
            <th>姓名</th>
            <th>年龄</th>
        </tr>
    </thead>
    <tbody>
        <tr>
            <td>xu</td>
            <td>18</td>
        </tr>
    </tbody>
    <tfoot></tfoot>
</table>
```
table上的 border-collapse
边框的折叠方式，默认为sequrate,分离的
td上的 colspan跨越列
rowspan跨越行
